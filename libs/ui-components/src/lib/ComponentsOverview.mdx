## ğŸ“– Usage

### Installation

```bash
npm install @true-tech-team/ui-components
```

### Basic Import

```tsx
import { Button, Input, Icon } from '@true-tech-team/ui-components';

function MyComponent() {
  return (
    <div>
      <Button variant="primary" size="md">
        Click me
      </Button>
      <Input placeholder="Enter text..." />
      <Icon name="check" size={24} />
    </div>
  );
}
```

### With Global Provider

Wrap your app with `GlobalProvider` to enable theming:

```tsx
import { GlobalProvider } from '@true-tech-team/ui-components';

function App() {
  return (
    <GlobalProvider themeConfig={{ mode: 'dark' }}>
      {/* Your app */}
    </GlobalProvider>
  );
}
```

### Category-Specific Imports

For better tree-shaking, import from specific categories:

```tsx
import { Button } from '@true-tech-team/ui-components/buttons';
import { Input, Select } from '@true-tech-team/ui-components/inputs';
import { Tooltip } from '@true-tech-team/ui-components/overlays';
```


### Theme Colors
Access theme colors via CSS custom properties:

```scss
background-color: var(--theme-primary);
color: var(--theme-text-on-primary);
border-color: var(--theme-border-primary);
```
